# renvault-staking

Staking module for the RenVault DeFi protocol on Stacks. Stake STX, earn rewards, and manage lock periods on-chain.

## Installation

```bash
npm install renvault-staking
```

## Features

- Stake and unstake STX with configurable lock periods
- Claim staking rewards from the reward pool
- Yield calculation utilities
- Rewards distribution management
- REST API for staking operations

## Usage

```js
const { stakingManager } = require('renvault-staking');
```
